# VidOps Webapp Frontend (React + TypeScript + Vite)

This frontend runs on **http://localhost:8090** and proxies **/api/** calls to the API Gateway on **http://localhost:8080**.

## Run (IDE-friendly)
1. Install dependencies (IntelliJ can detect `package.json` and offer an Install action):
   - `npm install`
2. Start dev server:
   - `npm run dev`

## Pages
- `/login`
- `/register`
